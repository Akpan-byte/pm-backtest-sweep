# Lead-to-Site Outreach Pipeline

A high-fidelity, serverless lead generation and outreach system.

## Overview
Automates the end-to-end flow:
1. **Scrape**: Intelligent website analysis using Jina AI.
2. **Generate**: High-fidelity, branded "Bento-style" dark-mode landing pages.
3. **Deploy**: Automatic deployment to public infrastructure.
4. **Outreach**: Personalized AI-generated emails via Gmail API + Instantly webhook.

## Structure
- `/lead-to-site`: Modal backend (compute)
- `/lead-to-site-ui`: Next.js frontend (dashboard)
